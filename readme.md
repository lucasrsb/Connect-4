## Pré-requisitos

- Docker instalado
- Python 3.8+
- Biblioteca `kazoo`

## Como executar

# 1. Inicie Zookeper e serviços auxiliares

docker-compose up -d

# 2. Instale as dependências Python

pip install kazoo

# 3. Execute o comando abaixo em dois terminais parados

python src/connect4.py

# 4. Divirta-se!
